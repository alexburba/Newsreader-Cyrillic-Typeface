# Newsreader Cyrillic

![Screenshot 2026-01-10 at 19 32 36](https://github.com/user-attachments/assets/8b9c2493-9155-43a9-9e6b-137cc868ed53)

Newsreader Cyrillic is a cyrillic fork of the [Newsreader](https://github.com/productiontype/Newsreader) typeface. It is a contemporary (until 2030) transitional sans-serif typeface. If you don't know what is it — that is Times New Roman, but much better.

- **Languages added:** Russian
- **Styles:** Display Regular, Display Light Italic
- **Kerning:** 30%
- **Plans (maybe):** BY, UA, Serbian and Bulgarian cyrillic + a proper kerning 
- **License:** SIL OFL 1.1
- **In Russian:** Newsreader Cyrillic — это кириллическая версия шрифта [Newsreader](https://github.com/productiontype/Newsreader). Newsreader — это современная (до 2030 года) переходная антиква. Если вы не знаете, что это — то, это Timews New Roman, но гораздо лучше. Пока почти готовы начертания Display Regular и Display Light Italic. Пока только русская кириллица. Потом добавлю BY, UA и сербскую кириллицу.
